# Music Preference Data Analysis

This project analyzes people's music preferences and listening patterns using data analysis techniques.

## Project Overview

The goal of this project is to identify trends in music preferences based on factors such as age, gender, listening habits, and favorite genres.

Through this analysis, I aimed to discover meaningful patterns and understand how different groups of people consume music differently.

## Skills Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Data Visualization
- Exploratory Data Analysis (EDA)

## Key Analysis Areas

- Most preferred music genres
- Listening time by age group
- Relationship between gender and genre preference
- Weekly listening habits
- Popular artists and songs trends

## Key Findings

- Younger users showed stronger preference for pop and hip-hop genres.
- Older groups tended to prefer ballad and classic genres.
- Listening time was generally higher during weekends.
- Genre preference showed noticeable differences across demographic groups.
